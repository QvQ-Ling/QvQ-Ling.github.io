printf 函数的格式说明：

%d：以 十进制（decimal）格式打印整数。

%o：以 八进制（octal）格式打印整数。

%x：以 十六进制（hexadecimal）格式打印整数。

在 C 语言中，最简单的数据类型（即基础数据类型）包括：

  1. 整型（int）：用于表示整数。
  2. 实型（float 和 double）：用于表示浮点数（实数）。
  3. 字符型（char）：用于表示单个字符。

单目运算符 ++（自增）和 --（自减）是 C 语言中的常用运算符

它们的运算对象可以是char型变量、int型变量和float型变量

char 类型（会将字符按其 ASCII 值进行自增或自减）

int 类型（整型变量）

float 类型（浮点型变量）

double 类型（虽然没有在选项中列出，但也是合法的）

设变量x为float型且已赋值，则以下语句中能将x中的数值保留到小数点后两位，并将第三位四舍五入的是。

A. x = x * 100 + 0.5 / 100.0; B. x = (x * 100 + 0.5) / 100.0; C. x = (int)(x * 100 + 0.5) / 100.0; D. x = (x / 100 + 0.5) * 100;

A. x = x * 100 + 0.5 / 100.0;

  • 这条语句的含义是：首先将 x 乘以 100，然后加上 0.5 / 100.0，这相当于加上一个非常小的值（0.005），它不会影响四舍五入到小数点后两位。
  • 问题：它并没有实现四舍五入到小数点后两位。它将 x 扩大 100 倍后增加一个极小的值 0.005，但这样做没有真正的四舍五入。
  • 结论：这个选项不正确。

B. x = (x * 100 + 0.5) / 100.0;

  • 这条语句的操作是：首先将 x 乘以 100，然后加上 0.5，接着再除以 100。这会先将 x 扩大 100 倍，然后加上 0.5，再除以 100。
  • 四舍五入解释：加上 0.5 后会使得小数点后第三位的数值大于等于 5 时产生进位，从而实现四舍五入。
  • 结论：这个选项是正确的，因为它可以有效地将 x 保留到小数点后两位，并进行四舍五入。

C. x = (int)(x * 100 + 0.5) / 100.0;

  • 这条语句的操作是：首先将 x 扩大 100 倍并加上 0.5，然后强制转换为整数 int，再除以 100。
  • 问题：强制转换为 int 会去掉小数部分，从而丢失四舍五入后的结果。最终得到的是截断结果，而不是四舍五入后的数值。
  • 结论：这个选项不正确，因为它会导致截断，而不是四舍五入。

D. x = (x / 100 + 0.5) * 100;

  • 这条语句的操作是：首先将 x 除以 100，然后加上 0.5，最后再乘以 100。
  • 问题：首先将 x 除以 100 会缩小 x，然后加上 0.5，这种操作并不直接有效地处理小数点后两位的四舍五入。
  • 结论：这个选项不正确，因为它并没有在正确的位置进行四舍五入操作。

设有定义“int k = 0;”，下列选项的4个表达式中与其他3个表达式的值不相同的是。

A. k++ B. k+=1 C. ++k D. k+1

我们需要分析这四个表达式在执行后的效果和最终的值是否相同。假设我们有变量 k = 0;。

A. k++

  • k++ 是 后置自增 运算符。首先会返回 k 当前的值（即 0），然后将 k 增加 1。
  • 执行完后，k 的值变为 1，但表达式的返回值是 0（即原始的值）。

B. k += 1

  • k += 1 是 加法赋值运算符，等同于 k = k + 1。
  • 这会将 k 的值增加 1，因此 k 的值变为 1，并且表达式的值也是 1。

C. ++k

  • ++k 是 前置自增 运算符。首先将 k 增加 1，然后返回增加后的值。
  • 执行完后，k 的值变为 1，且表达式的返回值也是 1。

D. k + 1

  • k + 1 是一个 算术表达式，它只是将 k 当前的值加上 1，但并不会改变 k 的值。
  • 执行完后，k 的值仍然是 0，而表达式的值是 1。

变量的初始值：

1. !(x + y)

  • 首先计算 x + y，即 3 + 4 = 7。
  • 然后计算 !(7)，! 是逻辑非运算符，它将一个非零值转换为 false（即 0），将零值转换为 true（即 1）。
  • 因为 7 是非零值，所以 !(7) 结果为 0。

2. z - 1

  • z 的值是 5，所以 z - 1 的值是 5 - 1 = 4。

3. y + z / 2

  • z / 2 进行的是整数除法，因为 z 和 2 都是整数类型，结果是 5 / 2 = 2（整数除法舍去小数部分）。
  • 然后计算 y + 2，即 4 + 2 = 6。

4. 逻辑运算符 &&

  • 现在我们将表达式拆成两个部分，分别是：

    • !(x + y) + z - 1 计算结果为 0 + 4 = 4
    • y + z / 2 计算结果为 6

    ：

    • 因为 4 和 6 都是非零值，所以 4 && 6 的结果为 1。

C语言优先级排序

以下是C语言常用运算符按优先级从高到低的排序：

1. 括号运算符：

  • ()：用于改变运算顺序。

2. 后缀运算符：

  • []（数组下标）
  • ()（函数调用）
  • .（成员访问，结构体中的点操作）
  • ->（成员访问，指针指向的结构体）
  • ++（后置递增）
  • --（后置递减）

3. 前缀运算符：

  • ++（前置递增）
  • --（前置递减）
  • +（一元加）
  • -（一元减）
  • !（逻辑非）
  • ~（按位取反）
  • &（取地址）
  • *（解引用，指针操作）
  • sizeof（大小运算符）

4. 乘法、除法、取模运算符：

  • *（乘法）
  • /（除法）
  • %（取模）

5. 加法、减法运算符：

  • +（加法）
  • -（减法）

6. 移位运算符：

  • <<（左移）
  • >>（右移）

7. 关系运算符：

  • <（小于）
  • <=（小于等于）
  • >（大于）
  • >=（大于等于）

8. 相等运算符：

  • ==（等于）
  • !=（不等于）

9. 按位与运算符：

  • &（按位与）

10. 按位异或运算符：

  • ^（按位异或）

11. 按位或运算符：

  • |（按位或）

12. 逻辑与运算符：

  • &&（逻辑与）

13. 逻辑或运算符：

  • ||（逻辑或）

14. 条件运算符（三元运算符）：

  • ? :（条件运算符）

15. 赋值运算符：

  • =（赋值）
  • +=（加法赋值）
  • -=（减法赋值）
  • *=（乘法赋值）
  • /=（除法赋值）
  • %=（取模赋值）
  • &=（按位与赋值）
  • ^=（按位异或赋值）
  • |=（按位或赋值）
  • <<=（左移赋值）
  • >>=（右移赋值）

16. 逗号运算符：

  • ,（逗号运算符，用于在一个表达式中执行多个子表达式）

运算符的结合性：

  • 从左到右结合：大部分运算符，如加法、减法、乘法等，都是从左到右计算的。
  • 从右到左结合：赋值运算符（=），以及条件运算符（?:）等。

优先级总结：

  1. 最先执行：括号运算符 () 和后缀运算符（[], (), ., ->, ++, --）的优先级最高。
  2. 其次是：一元运算符（如 ++, --, !, ~）和乘法、除法、取模运算符。
  3. 再往下是：加法、减法、移位、关系运算符、按位与、异或、或等运算符。
  4. 最后执行：赋值运算符和逗号运算符。

设有定义“int a = 2, b = 3, c = 4;”，则下列选项中值为0的表达式是____。

A.

(!a == 1) && (!b == 0)

B.

(a < b) && !c || 1

C.

a && b

D.

a || (b + b) && (c - a)

需要分析每个表达式的值，给定初始值 a = 2, b = 3, c = 4，并找出值为0的表达式。

A. (!a == 1) && (!b == 0)

  • !a 等于 0（因为 a = 2，非零的数为真，取反后为假，即 0）。
  • !b 等于 0（同理，b = 3）。
  • !a == 1 是 0 == 1，即假。
  • !b == 0 是 0 == 0，即真。
  • 因为 && 运算符要求左右两侧都为真，所以 false && true = false，结果为 0。

选项 A 的值为 0。

B. (a < b) && !c || 1

  • a < b 是 2 < 3，即真，值为 1。
  • !c 是 !4，因为 c = 4 非零，取反后为 0。
  • 1 && 0 等于 0。
  • 0 || 1 最终结果为 1。

选项 B 的值为 1，不是 0。

C. a && b

  • a = 2，b = 3，两个都非零，逻辑与 && 运算符返回真。
  • 2 && 3 等于 1。

选项 C 的值为 1，不是 0。

D. a || (b + b) && (c - a)

  • a = 2，因此 a 为真，逻辑或 || 运算符会直接返回真，不再计算右侧表达式。
  • 所以，整个表达式 a || (b + b) && (c - a) 的值为 2，即真。

选项 D 的值为 1，不是 0。

步骤 1: c2 - c1

  • c1 = 'b'，其 ASCII 值为 98。
  • c2 = 'e'，其 ASCII 值为 101。
  • c2 - c1 = 101 - 98 = 3。

所以，c2 - c1 的结果是 3。

步骤 2: c2 - 'a' + 'A'

  • c2 = 'e'，其 ASCII 值为 101。
  • 'a' 的 ASCII 值是 97，'A' 的 ASCII 值是 65。
  • c2 - 'a' = 101 - 97 = 4。
  • 4 + 'A' = 4 + 65 = 69。

因此，c2 - 'a' + 'A' 的结果是 69，对应的字符是 'E'（因为 ASCII 值为 69 的字符是 'E'）。

步骤 3: 输出分析

  • %d 会输出一个整数，c2 - c1 的结果是 3，所以输出的第一个值是 3。
  • %c 会输出一个字符，c2 - 'a' + 'A' 的结果是 69，对应的字符是 'E'，所以输出的第二个值是 'E'。
