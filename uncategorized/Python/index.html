3、from itertools import permutations

from itertools import permutations 导入的是 Python 标准库 itertools 中的 permutations 函数，它用于生成可迭代对象（如列表、字符串等）中元素的所有可能排列组合。

基本用法

permutations(iterable, r) 有两个参数：

  • iterable：要生成排列的可迭代对象（如列表、字符串、元组等）
  • r（可选）：生成的排列长度，默认等于 iterable 的长度（即全排列）

返回值是一个迭代器，每个元素是由 r 个元素组成的元组，表示一种排列。

示例代码

特点

  • 生成的排列是有序的（不同顺序视为不同排列）
  • 不会包含重复元素（前提是原 iterable 中没有重复元素）
  • 返回的是迭代器，需要通过循环或转换为列表（如 list(permutations(...))）来获取所有结果
